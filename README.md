# ワシソダ分類er / SodaCH Works Classifier

## 概要 / Overview

**曽田すかい@ワシソダch** の配信・動画でプレイする作品を管理する、**ワシソダCH専用の作品データベース**です。<br>
各動画・配信に対して「作品名」と「作品公式サイト URL」を管理し、[SodaCH Discord Notifier](https://github.com/xiang4ri4kui2/SodaCH-discord-notifier) へ提供します。<br>
<br>
技術的には、作品分類ルールをまとめた `worksMaster.json` と、その辞書でカバーできない配信・動画（未分類動画）を検出し、GitHub Issue を自動作成するツールを提供しています。<br>
また、[SodaCH Discord Notifier](https://github.com/xiang4ri4kui2/SodaCH-discord-notifier) が Discord 通知の Embed に表示する「作品名」と「作品公式サイト URL」の判定元データも提供します。<br>
<br>
This repository serves as a dedicated work database for the YouTube channel **"曽田すかい@ワシソダch"**, managing the games and other works featured in its videos and live streams.<br>
It maintains the work title and official website URL for each video or live stream, and provides this information to [SodaCH Discord Notifier](https://github.com/xiang4ri4kui2/SodaCH-discord-notifier).<br>
<br>
Technically, this repository provides `worksMaster.json`, which contains the rule-based work classification data, along with a tool that detects unclassified videos not yet covered by the dictionary and automatically creates GitHub Issues for them.<br>
It also serves as the source of the work title and official website URL displayed in Discord notification embeds generated by [SodaCH Discord Notifier](https://github.com/xiang4ri4kui2/SodaCH-discord-notifier).<br>

### 対象チャンネル / Target Channel

**曽田すかい@ワシソダch**<br>
https://www.youtube.com/@BabiSodaSky

## 機能 / Features

本リポジトリは、以下の機能を持ちます。

* ワシソダCH専用の作品データベース（`worksMaster.json`）の管理
* 新規の配信・動画（前回チェック以降の差分）を取得し、作品データベースに未登録の物を検出
* 未登録の配信・動画が見付かった場合、候補文字列（Youtube のタイトルとタグ情報）を基に、JSON形式の候補データを含む GitHub Issue を自動作成

This repository provides the following features:

* Maintains the dedicated work database for the YouTube channel "曽田すかい@ワシソダch" (`worksMaster.json`)
* Fetches newly published videos and live streams since the last check and detects content that cannot be classified by the work database
* Automatically creates a GitHub Issue containing candidate JSON data generated from the YouTube title and tag information when unclassified videos or live streams are detected

## 実行間隔 / Monitoring Schedule

負荷分散のため、GitHub Actions は **毎日 04:17 に 1 回**実行しています。

To distribute system load, the workflow runs **once daily at 04:17 JST**.
